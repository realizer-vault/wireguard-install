# OpenVPN & WireGuard installer



**This project is a bash script that aims to setup a OpenVPN & [WireGuard](https://www.wireguard.com/) VPN on a Linux server, as easily as possible!**

Download and execute the script. Answer the questions asked by the script and it will take care of the rest.

```bash
curl -O https://raw.githubusercontent.com/realizer-vault/OVPN-WG-Installer/master/VPN-Installer.sh
chmod +x VPN-Installer.sh
./VPN-Installer.sh
```

OpenVPN Menu
```bash
ovpn.sh
```
WireGuard Menu
```bash
wg.sh
```
Move *.ovpn & *.conf to client folder
```bash
move.sh
```
